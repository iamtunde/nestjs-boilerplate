

## Description

A NestJS RESTful API boilerplate
<br>
### Features
The current stage of the boiler plate has the following features.
- Authentication
  - Sign up
  - Sign in


## Configuration
```bash
# create .env file from example file
$ cp .env.example .env
```
<p>Modify the .env file with your preferred configuration</p>


## Migrations & Seeds
```bash
# run migrations on the database
$ npm run db:migrate

# run database seed data
$ npm run db:seed:all
```
## Running the app
```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```
